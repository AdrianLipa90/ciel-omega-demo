# Analogy Registry

## Purpose

This registry maps formal Omega / CIEL concepts to mnemonic images and teaching analogies.

Each entry should preserve four layers:

- **formal term**
- **plain explanation**
- **mnemonic image**
- **warning / limit of the analogy**

---

## Registry

### 1. Truth Attractor
- **Formal term:** Truth Attractor
- **Plain explanation:** The system tends toward a state of minimum distortion.
- **Mnemonic image:** A mountain valley pulling every rolling marble toward the same lowest basin.
- **Limit:** Truth is not just “whatever most marbles touch first”; false local minima can exist.

### 2. Node
- **Formal term:** Node
- **Plain explanation:** A local state-holder in the system.
- **Mnemonic image:** A lamp-post in a foggy city, each one holding a little light.
- **Limit:** A node is not the whole system and does not define truth by itself.

### 3. White Thread
- **Formal term:** White Thread
- **Plain explanation:** A live relation or dependency linking nodes/layers.
- **Mnemonic image:** A glowing string connecting two lanterns in the dark.
- **Limit:** A thread is not just decoration; it represents real coupling.

### 4. Identity Attractor
- **Formal term:** Identity Attractor
- **Plain explanation:** The organizing center that keeps the system coherent as one system.
- **Mnemonic image:** The pole in the middle of a tent holding all the ropes in one tension pattern.
- **Limit:** Identity is not the same as a chat window or a single model.

### 5. Memory Topology
- **Formal term:** Memory Topology
- **Plain explanation:** The structured arrangement of memory channels and their dependencies.
- **Mnemonic image:** A spider web where each strand changes the tension of all the others.
- **Limit:** Memory is not just storage; it is structured relation.

### 6. Publication Boundary
- **Formal term:** Publication Boundary
- **Plain explanation:** The layer that separates private/internal content from public/exported content.
- **Mnemonic image:** A castle gate deciding what leaves the city and what stays inside.
- **Limit:** A boundary is not censorship by default; it is separation and control.

### 7. Evidence
- **Formal term:** Evidence Layer
- **Plain explanation:** Logs, tests, provenance, and audit signals that support or weaken claims.
- **Mnemonic image:** A trail of footprints showing where someone really walked.
- **Limit:** Evidence is not identical with truth, but it constrains convergence.

### 8. Kernel
- **Formal term:** Kernel
- **Plain explanation:** The active computational engine performing state updates.
- **Mnemonic image:** A mill wheel turning the flow of water into motion.
- **Limit:** The kernel executes; it does not by itself explain meaning.

### 9. Constraint
- **Formal term:** Constraint
- **Plain explanation:** A condition that limits valid motion or valid states.
- **Mnemonic image:** Rails keeping a train on a meaningful path.
- **Limit:** Constraints do not destroy freedom; they define coherent possibility.

### 10. Consensus
- **Formal term:** Convergent Consensus
- **Plain explanation:** Agreement that appears when many nodes are pulled toward the same attractor.
- **Mnemonic image:** Many compass needles settling toward one magnetic direction.
- **Limit:** Consensus is not truth if it comes from imitation instead of attraction to the real basin.
