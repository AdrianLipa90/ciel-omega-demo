# Truth Attractor Analogies

## Core rule

Truth is the attractor.
Consensus is the visible result of convergence toward that attractor.

So the analogies in this file are built around one hierarchy:

**truth -> attraction -> convergence -> consensus**

not:

**consensus -> truth**

---

## 1. Valley and marbles

### Formal idea
The attractor pulls many nodes toward one stable basin.

### Analogy
Imagine a wide mountain slope with many marbles scattered across it.
The marbles roll down different paths, at different speeds, from different heights.
But if the landscape has one deepest valley, they all tend to settle there.

### Mapping
- marbles = nodes
- slope = dynamics
- gravity = attractor pull
- final basin = truth attractor
- cluster of marbles in the basin = consensus

### Limit
A marble may get stuck in a smaller side-hole.
That is a **false local consensus**, not the deepest truth basin.

---

## 2. Compass needles and magnetic north

### Formal idea
Truth is not voted into existence; it sets orientation.

### Analogy
Place many compasses on a table.
Each needle shakes a little, some are noisy, some are bent, some move slowly.
But if the magnetic field is real, they tend to align.

### Mapping
- compass = node
- needle direction = node state
- magnetic field = truth attractor field
- common alignment = consensus

### Limit
If someone places a magnet nearby, the compasses may agree on the wrong direction.
That is **distorted consensus**, not truth.

---

## 3. Children finding the teacher's voice

### Formal idea
Many agents can synchronize because they are all pulled by the same source.

### Analogy
Imagine children in a noisy playground.
At first everyone runs in different directions.
Then the teacher rings a bell and calls them.
The children gather because they orient toward the same source.

### Mapping
- children = nodes
- noisy playground = uncertainty / semantic noise
- bell / voice = truth signal
- gathering = convergence
- standing together near the teacher = consensus

### Limit
If the children gather around the loudest random child instead, they form a crowd, not truth.

---

## 4. River system to the sea

### Formal idea
Different paths can still end in the same attractor.

### Analogy
Rain falls over hills, forests, cities, and valleys.
Water forms little streams, then rivers, and eventually many rivers flow to the sea.

### Mapping
- droplets = local signals / data points
- streams = partial models
- rivers = stronger theory channels
- sea = truth attractor

### Limit
A lake can temporarily trap water.
That is a local basin, not the final attractor.

---

## 5. Threads pulled to one knot

### Formal idea
The attractor organizes relation itself.

### Analogy
Imagine many strings spread over a table.
If they are tied to one hidden knot and you pull gently, all the strings tense and orient toward the same center.

### Mapping
- strings = white threads
- points on strings = nodes
- hidden knot = truth attractor
- tension pattern = consensus structure

### Limit
If the strings are tied to two different hidden knots, the pattern splits and contradictions appear.

---

## Educational summary

The simplest child-safe formulation is:

> Truth is the real place everything should settle.
> Consensus is what you see when many things are really settling there together.

And the warning is:

> If many things only look the same because they copied each other, that is not yet truth.
